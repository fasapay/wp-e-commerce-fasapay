[English]

COPYING/UPLOADING THE FILES

1. Ensure you have WordPress and WP eCommerce installed on your website
2. Install fasapay plugins
3. The FasaPay plugins will automaticaly appear in the admin area under
   Settings -> fasapay plugins setting
4. Select wp-ecomerce radio button then press Add button to shoping card plugins

CONFIGURING THE GATEWAY

1. The FasaPay Payment Gateway will automaticaly appear in the admin area under
   Settings -> Store -> payments
2. Configure the gateaway by ticking it's check box and click Update
3. Now Configure Fasapay by hovering it and click edit
4. Fill out the details on the right hand side.
5. Now you can test the checkout and you will find FasaPay as one of the Payment options. 

[Indonesia]

MENYALIN/UPLOAD FILE

1. Pastikan anda menggunakan WordPress dan Menginstall WP eCommerce di website anda.
2. Insstal plugin fasapay
3. Plugin FasaPay akan otomatis muncul di area admin di bawah
   Settings -> fasapay plugins setting
4. pilih radio button wp-ecomerce kemudian tekan tombol Add to shoping card plugins

KONFIGURASI GATEWAY

1. FasaPay Payment Gateway akan otomatis muncul di admin area pada 
   Settings -> Store -> payments

2. Konfigurasi FasaPay Payment Gateway dengan mencentang check box disampingnya. 
   Kemudian Klik Update

3. Sekarang Konfigurasi FasaPay dengan mengarahkan mouse diatasnya dan klik edit
4. Isi data yang ada disamping kanan dengan benar dan klik update
5. Sekarang anda dapat melakukan test checkout dan akan menemukan 
   FasaPay sebagai salah satu metode pembayaran
